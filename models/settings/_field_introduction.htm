IndexNow is a protocol that notifies search engines whenever a website's content changes. Configure your API key below and optionally enable auto-submission of your sitemap.

How it works:
1. Click Generate & Save API Key — a key will be auto-created if needed
2. The plugin automatically serves your verification file at /[key].txt
3. Click Submit Sitemap to IndexNow Now to notify Bing
4. Or enable Auto submit to automatically notify IndexNow on sitemap changes
